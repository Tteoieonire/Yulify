# Yulify
국악 율명을 nwctxt 파일에 가사로서 덧붙입니다. nwctxt 파일은 작곡 프로그램 Noteworthy Composer로 작성할 수 있습니다.

Attaches yullyeo reading of notes in nwctxt file as a lyric.
